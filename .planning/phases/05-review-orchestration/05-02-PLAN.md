## Tasks

### Task 1: Implement `getPrDiff` in `src/pipeline-io.ts`

```typescript
export interface PrDiffResult {
  diff: string;
  stats: DiffStats;
  prNumber: number;
  branch: string;    // head.ref — used in session slug
  repoSlug: string;  // "owner/repo" — for INPUT.md metadata
}

export async function getPrDiff(
  prNumber: number,
  token: string,
  repoSlug: string,
): Promise<PrDiffResult>
```

- Two GitHub API calls:
  1. `GET /repos/{owner}/{repo}/pulls/{pull_number}` (JSON) → get `head.ref` branch name
  2. `GET /repos/{owner}/{repo}/pulls/{pull_number}` with `Accept: application/vnd.github.v3.diff` → raw diff
- Auth: `Authorization: Bearer {token}`
- Error handling:
  - 404 → `PR #{prNumber} not found in {repoSlug}. Check the PR number and GITHUB_TOKEN.`
  - 401/403 → `GitHub authentication failed. Set GITHUB_TOKEN with repo read access.`
  - Empty diff → `PR #${prNumber} diff is empty. It may have no file changes or may be already merged.`
- Runs `preprocessDiff` on raw diff (same as local — strips lock files, binaries)
