## Goal

Replace prose-stub command files with thin shell invocations of the Node.js CLI. Add collision-safe session slugs and file-existence verification at pipeline handoffs. Verify local review success criteria.

## Success Criteria Covered

| Criterion | How verified |
|-----------|-------------|
| 1. `/review local` → `REPORT.md` | Command files invoke CLI; CLI runs full pipeline |
| 3. `--focus security` → emphasis reflected | `$ARGUMENTS` passes through; reviewer prompt suppresses non-focus dimensions |
| 4. Empty diff → clear error | Already in index.ts; verify test coverage |
| 5. Missing step file → loud failure | `verifyFileExists` throws with label before each handoff |

## Tasks

### Task 1: Update OpenCode command template

`src/templates/opencode-review-local.md` — replace prose steps with:
```markdown
---
description: Run rms code review on local git diff (staged + unstaged changes)
argument-hint: [--focus <area>]
subtask: true
---
!node dist/index.js review-local $ARGUMENTS
```

### Task 2: Update Cursor command template

`src/templates/cursor-review-local.md` — Cursor has no `!` shell injection; use prose-with-CLI-call pattern:
```markdown
---
description: Run rms code review on local git diff (staged + unstaged changes)
---
Run the following command in the terminal and report its output to the user:

node dist/index.js review-local $ARGUMENTS

Then open the REPORT.md file it creates in `.reviews/` and present the findings.
```

### Task 3: Sync installed command files

Update `.opencode/commands/review-local.md` and `.cursor/commands/review-local.md` to match templates.

### Task 4: Add collision-safe slug for local reviews

`src/index.ts`: change `createSession(projectRoot, 'local')` → `createSession(projectRoot, \`local-${nanoid(4)}\`)`
Add `import { nanoid } from 'nanoid';` to `src/index.ts`.

### Task 5: Add `verifyFileExists` helper

`src/pipeline-io.ts`: add:
```typescript
export async function verifyFileExists(filePath: string, label: string): Promise<void> {
  try {
    await stat(filePath);
  } catch {
    throw new Error(`[rms] Pipeline error: ${label} not found at ${filePath}. Previous step may have failed.`);
  }
}
```

Call in `src/index.ts` after each step:
1. After `runReviewer` → before `runValidator`: `await verifyFileExists(result.reviewerMdPath, 'REVIEWER.md')`
2. After `runValidator` → before `runWriter`: `await verifyFileExists(validatorResult.validatorMdPath, 'VALIDATOR.md')`

### Task 6: Tests

- Unit test for `verifyFileExists`: file exists → resolves; missing → throws with label in message
- Test for collision-safe slug: two sessions in same test run produce different `reviewId`s

### Task 7: Build + type check

`npm run build` + `npm test` both pass.
