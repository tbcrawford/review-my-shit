BEFORE (node:test style):
```typescript
import { test, describe } from 'node:test';
import assert from 'node:assert/strict';

```

AFTER (vitest style):
```typescript
import { test, describe, expect, beforeAll, afterAll, beforeEach, afterEach, vi } from 'vitest';
// Only import what you use — don't import unused names
```

ASSERTION CHEATSHEET (use exactly these mappings):
| node:assert | vitest expect |
|-------------|---------------|
| `assert.strictEqual(a, b)` | `expect(a).toBe(b)` |
| `assert.equal(a, b)` | `expect(a).toBe(b)` |
| `assert.deepStrictEqual(a, b)` | `expect(a).toEqual(b)` |
| `assert.ok(x)` | `expect(x).toBeTruthy()` |
| `assert.ok(x, 'msg')` | `expect(x).toBeTruthy()` (drop msg — vitest auto-formats) |
| `assert.match(str, /re/)` | `expect(str).toMatch(/re/)` |
| `assert.throws(() => fn())` | `expect(() => fn()).toThrow()` |
| `assert.throws(() => fn(), /msg/)` | `expect(() => fn()).toThrow(/msg/)` |
| `await assert.rejects(promise)` | `await expect(promise).rejects.toThrow()` |
| `await assert.rejects(promise, /msg/)` | `await expect(promise).rejects.toThrow(/msg/)` |

LIFECYCLE HOOKS:
| node:test | vitest |
|-----------|--------|
| `before(fn)` (top-level) | `beforeAll(fn)` |
| `after(fn)` (top-level) | `afterAll(fn)` |
| `beforeEach(fn)` | `beforeEach(fn)` (same name) |
| `afterEach(fn)` | `afterEach(fn)` (same name) |
