  <files>src/session.ts, src/session.test.ts</files>
  <behavior>
    - `ensureReviewsDir(projectRoot: string): Promise<void>` — creates `.reviews/` if absent, then checks if `.gitignore` at projectRoot already contains `.reviews/`; if not, appends `\n.reviews/` to `.gitignore` (creates `.gitignore` if it doesn't exist). Idempotent — safe to call multiple times.
    - `createSession(projectRoot: string, slug: string): Promise<SessionInfo>` — calls `ensureReviewsDir`, then creates `.reviews/YYYY-MM-DD-<slug>/` where date is today in UTC. Returns `{ sessionDir, reviewId, slug, timestamp }`.
    - `SessionInfo` type: `{ sessionDir: string; reviewId: string; slug: string; timestamp: string }`
    - `reviewId` is the folder name: `YYYY-MM-DD-<slug>` (e.g., `2026-04-04-feat-auth`)
    - Slug sanitization: lowercase, replace spaces and slashes with `-`, strip non-alphanumeric except `-`
    - Tests verify: `ensureReviewsDir` creates `.reviews/`, appends to `.gitignore` when missing, is idempotent when `.reviews/` already in `.gitignore`; `createSession` creates correct folder structure, returns correct `reviewId` format
  </behavior>
  <action>
    Create `src/session.ts` exporting `SessionInfo` type, `ensureReviewsDir`, and `createSession`.

    Slug sanitization: `slug.toLowerCase().replace(/[^a-z0-9-]/g, '-').replace(/-+/g, '-').replace(/^-|-$/g, '')`.
